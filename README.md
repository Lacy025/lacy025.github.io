# Poker machine with 53 cards in HTML/CSS/Javascript.

Q = Credit + 100   
5 = Bet + 1   
Space = Autohold On/Off   
Enter = Deal cards   
1,2,3,4,5 = Hold cards   
Space = Reset hold cards   
Enter = Double or nothing   
Left CTRL = Tief   
Right CTRL = Hoch   
5 = Take a half   
Space = Take all   
End - Cashout   

Enjoy !
